# CSIR JSA Typing Test Practice

CSIR JSA Typing Test Practice tool is designed for candidates preparing for CSIR Junior Secretariat Assistant typing exams.

## Features

- CSIR JSA typing practice
- English and Hindi typing practice
- Timer based typing test
- Gross speed and net speed result
- Accuracy and error tracking
- Instant result screen
- Exam-style typing interface

## Typing Rules

- Exam: CSIR Junior Secretariat Assistant
- Language: English & Hindi
- English Speed Target: 35 WPM
- Hindi Speed Target: 30 WPM
- Official Style Duration: 10 Minutes
- Practice Modes: 2 Min, 5 Min, 7 Min
- Recommended Accuracy: 90%+
